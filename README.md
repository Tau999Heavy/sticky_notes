# Sticky Notes Application

A Django-based sticky notes application that allows users to create, edit, and delete notes.

## Features

- Create notes
- Edit notes
- Delete notes
- View notes
- Custom sticky note themed interface
- Unit tests

## Running the Project

1. Create a virtual environment

python -m venv venv

2. Activate the virtual environment

venv\Scripts\activate

3. Install dependencies

pip install -r requirements.txt

4. Run the application

python manage.py runserver
